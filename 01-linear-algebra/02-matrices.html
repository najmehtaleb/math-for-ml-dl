Square Matrix

Symmetric Matrix

Triangular Matrix

    [1,2,3],
    [1,2,3],
    [1,2,3]])

Sparse Matrices

Sparse Matrices in Python

A = array([
    [1,0,0,1,0,0],
    [0,0,2,0,0,1],
    [0,0,0,2,0,0]])
print(A)
[[1 0 0 1 0 0]
 [0 0 2 0 0 1]
 [0 0 0 2 0 0]]
S = csr_matrix(A)
print(S)
  (0, 0)    1
  (0, 3)    1
  (1, 2)    2
  (1, 5)    1
  (2, 3)    2
B = S.todense()
print(B)
[[1 0 0 1 0 0]
 [0 0 2 0 0 1]
 [0 0 0 2 0 0]]
sparsity = 1.0 - count_nonzero(A) / A.size
print(sparsity)
0.7222222222222222
